== Description ==

TimeSlotBooker is a lightweight, easy to use WordPress appointment calendar, online scheduling plugin that lets your customers book appointments with you. 

###Lite Version Features###
* Configure time slots, any start time, end time, duration for a regular weekly schedule or custom dates.
* Quick admin booking management.
* Simple, unobtrusive front end that smoothly fits into almost any theme.
* Customizable email notifications.
* Customizable options.

###Pro Version Features###

* __Multiple Calendars__ to manage bookings for various providers or resources.
* __Custom Fields__ to collect more information from your booking customers.
* __Online Payments__ coming soon.
* __SMS Text Messages__ coming soon.

== Installation ==

1. After unzipping, upload everything in the `timeslotbooker` folder to your `/wp-content/plugins/` directory (preserving directory structure).

2. Activate the plugin through the 'Plugins' menu in WordPress.

== Upgrade Notice ==
The upgrade is simply - upload everything up again to your `/wp-content/plugins/` directory, then go to the Locatoraid menu item in the admin panel. It will automatically start the upgrade process if any needed.
